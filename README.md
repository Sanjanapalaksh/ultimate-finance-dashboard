The Ultimate Finance Dashboard is a free, interactive web app that helps you track income, categorize expenses, set savings goals, and visualize spending patterns. It also provides smart suggestions on which expenses to reduce so you can save more efficiently.

💡 Features
User Authentication: Register and login securely

Salary & Expense Tracking: Input monthly salary and categorize expenses (Rent, Food, Transport, Entertainment, Miscellaneous)

Savings Goal: Set monthly savings target and track progress

Dashboard Metrics: Summary cards for Salary, Total Expenses, and Remaining Budget

Progress Bars: Visual representation of savings vs goal and budget usage

Charts & Visualization: Bar chart for expense distribution

Smart Suggestions: Alerts when expenses exceed budget and recommendations for expense reduction

Professional UI: Gradient background, cards, sidebar, and modern layout for a polished look

Responsive Design: Works well on desktop and tablet screens
